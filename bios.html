IL BIOS



E’ un programma incorporato nei computer che risiede su un chip della
scheda madre,gestisce l’avvio del sistema operativo all’accensione del computer.





  • Post

    Quando si accende il computer, il BIOS si avvia ed esegue prima di tutto un test
    per verificare che tutte le componenti hardware del tuo computer
    (gli hard disks interni, la scheda audio, la scheda video, la tastiera etc) siano connesse e funzionino correttamente.

    • Se il test è positivo e non ci sono problemi, il BIOS carica il bootloader,
      la procedura di ricerca e di avvio del sistema operativo
    Il bootloader, una volta individuato il tuo sistema operativo carica il sistema
    e nel giro di poco si vedrà la schermata di login a Windows (o il tuo desktop).

  • Accesso al bios

    Per accedere al bios basta riavviare il computer e, non appena compare la schermata
    di accensione del PC,premere ripetutamente il tasto necessario o la giusta
    combinazione di tasti per accedere al BIOS



  • I comandi di Linux:


    1) ls
    Serve a visualizzare il contenuto di una directory;

    2) cd
    Serve a spostarsi da una directory ad un’altra;

    3) mkdir
    Serve a creare una directory/cartella;


    4)rmdir
    Serve a eliminare una directory/cartella;

    5) >*nome*
    Serve a creare un file;


    6) rm
    Serve a eliminare un file;

    7) gedit
    Serve a modificare un file(testo);

    8) sudo mv *nome* -t *destinazione*
    Serve a spostare una directory/cartella o un file in una diversa directory;
    9) sudo mv *nome* -t *nuovo nome*
    Serve a rinominare le directory/cartelle e i file;




  • I comandi di windows:


    1) chiper
    Serve a sovrascrivere lo spazio disco libero in Windows in modo che i file cancellati col cestino non siano più recuperabili.
    Es.:
    "cipher /w:c" - pulisce l'unità disco C.

    2) driverquery
    Serve a fare la lista dei driver del computer con tutti i dettagli.

    3) shutdown
    Questo comando spegne il computer.
    Il comando “shutdown /r /o”, invece riavvia il PC e lancia il menu di opzioni avanzate di avvio
    per accedere alla modalità provvisoria e all'utilità di recupero.


    4) sfc (System File Checker) serve a fare il controllo dei file di sistema per correggere errori di file non trovati o corrotti.
    “sfc /scannow” - serve a rilevare file danneggiati o mancanti e sostituirli con file buoni.



    5) tasklist’ Questo comando serve a vedere l'elenco aggiornato di tutte le attività in esecuzione sul PC.
    Es.: “tasklist-svc” - mostra i servizi relativi a ciascuna attività.
    “tasklist-v” - fornisce maggiori dettagli su ogni processo.
    “tasklist-m” - individua i file dll con processi attivi.
    Questi comandi sono utili per la risoluzione avanzata dei problemi o per rintracciare manualmente malware.


    6) ipconfig È il comando per sapere quale indirizzo IP ha il computer collegato in rete.
    Se si usa un router si troverà l'indirizzo di rete locale del router.
    Es.: “ipconfig /release” seguito da “ipconfig /renew” - per chiedere un nuovo indirizzo IP, utile per ripristinare internet.
    “ipconfig /flushdns” - per aggiornare la risoluzione dei nomi DNS.
    “ipconfig /all” - mostra tutti gli indirizzi di rete del computer.


    7) netstat “netstat-an” - si ottiene la lista delle porte di rete aperte con i relativi indirizzi IP.
    Utile per cercare e risolvere problemi di malware o collegamenti a Internet dei software.


    8) ping Questo comando serve a controllare se il computer è connesso a Internet o se riesce a collegarsi a un dispositivo di rete.
    Oppure “ping indirizzo ip” del router per verificare se c'è collegamento della rete LAN.



    9) powercfg È un comando per la gestione dell'energia. È utile a verificare eventuali anomalie di sistema che possono aumentare il consumo di energia.
    Es.:
    "powercfg /hibernate on" e “powercfg /hibernate off” - per gestire la modalità di ibernazione.
    "powercfg /a" - per visualizzare gli stati di risparmio energetico.
    “powercfg /devicequery s1_supported” - per vedere la lista di dispositivi che possono
    risvegliare il computer dallo stato di sospensione.
    “Powercfg /lastwake” - dice qual è l'ultimo dispositivo che ha svegliato il PC da uno stato di sospensione.
    “powercfg /energy” può essere usato per avere una relazione sul consumo di energia del PC.
